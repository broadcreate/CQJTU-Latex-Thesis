# 重庆交通大学本科毕业论文格式要求（已填充）

**基于《重庆交通大学本科毕业论文(设计)模板》（2024更新版）**

---

## 1. 页面设置

- [x] **纸张大小**：A4 (210mm × 297mm)
- [x] **页边距**：
  - 上边距：**2.5cm**
  - 下边距：**2.0cm**
  - 左边距：**3.0cm**
  - 右边距：**2.0cm**
- [x] **装订线**：0cm
- [x] **页眉距边界**：1.0cm
- [x] **页脚距边界**：1.0cm
- [x] **打印方式**：单面打印或双面打印均可

---

## 2. 封面信息

- [x] **需要包含的字段**：
  - 校徽和校名（顶部居中）
  - 论文类型："本科毕业论文（设计）"
  - 题目（中文，可多行）
  - 学院
  - 专业
  - 学生姓名
  - 学号
  - 指导教师
  - 完成时间
  - 底部校名：重庆交通大学 / CHONGQING JIAOTONG UNIVERSITY

- [x] **封面格式要求**：
  - **校徽位置和大小**：顶部居中，宽度约10cm
  - **校名**：校徽下方，重庆交通大学 / CHONGQING JIAOTONG UNIVERSITY
  - **论文类型**："本科毕业论文（设计）"，二号黑体，居中
  - **题目**："题目："三号黑体，内容三号黑体，带下划线
  - **信息表格**：三号楷体，左对齐，带下划线
  - **底部校名**：三号宋体加粗 / 四号英文

---

## 3. 中英文摘要

### 中文摘要
- [x] **标题**："摘要"，三号黑体，居中
- [x] **正文字体字号**：小四号宋体
- [x] **行距**：1.5倍行距
- [x] **关键词格式**：
  - "关键词："小四号黑体
  - 关键词内容小四号宋体
  - 4-6个关键词
  - 关键词之间用分号";"分隔

### 英文摘要
- [x] **标题**：
  - 论文英文题目：三号Times New Roman加粗，居中
  - "Abstract"：三号Times New Roman加粗，居中
- [x] **正文字体字号**：小四号Times New Roman
- [x] **行距**：1.5倍行距
- [x] **关键词格式**：
  - "Key Words: "小四号Times New Roman加粗
  - 关键词内容小四号Times New Roman
  - 关键词之间用分号";"分隔
  - 末尾不加标点

---

## 4. 目录

- [x] **目录标题**："目录"，三号黑体，居中
- [x] **一级标题格式**：小四号宋体，章号+章标题，右侧页码，中间用点连接
- [x] **二级标题格式**：小四号宋体，缩进2字符，节号+节标题，右侧页码
- [x] **三级标题格式**：小四号宋体，缩进4字符，小节号+小节标题，右侧页码
- [x] **页码格式**：小四号宋体，阿拉伯数字
- [x] **插图/表格索引**：根据需要可选添加

---

## 5. 正文格式

### 章标题（一级标题）
- [x] **编号格式**："第1章"、"第2章"……（阿拉伯数字）
- [x] **字体**：黑体
- [x] **字号**：三号
- [x] **对齐方式**：居中
- [x] **章前换页**：每章均从新的一页开始

### 节标题（二级标题）
- [x] **编号格式**："1.1"、"1.2"……
- [x] **字体**：黑体
- [x] **字号**：四号
- [x] **对齐方式**：左对齐

### 小节标题（三级标题）
- [x] **编号格式**："1.1.1"、"1.1.2"……
- [x] **字体**：黑体
- [x] **字号**：小四号
- [x] **对齐方式**：左对齐

### 四级标题
- [x] **编号格式**："(1)"、"(2)"……或"①"、"②"……
- [x] **字体**：黑体
- [x] **字号**：小四号

### 正文
- [x] **字体**：宋体（中文）/ Times New Roman（英文、数字）
- [x] **字号**：小四号
- [x] **行距**：1.5倍行距
- [x] **段前距**：0磅
- [x] **段后距**：0磅
- [x] **首行缩进**：2字符

---

## 6. 图表公式

### 图题
- [x] **字体字号**：五号黑体
- [x] **位置**：图下方，居中
- [x] **编号格式**："图2-1"（章号-图序号）
- [x] **格式**："图2-1 图题内容"

### 表题
- [x] **字体字号**：五号黑体
- [x] **位置**：表上方，居中
- [x] **编号格式**："表2-1"（章号-表序号）
- [x] **格式**："表2-1 表题内容"
- [x] **表格推荐**：使用三线表

### 公式
- [x] **编号格式**：(章号-公式序号)，如：(2-1)
- [x] **编号位置**：公式右侧行末
- [x] **公式排版**：居中，与编号之间不加虚线
- [x] **公式字体**：Times New Roman

---

## 7. 参考文献

- [x] **标题**："参考文献"，三号黑体，居中
- [x] **正文字体字号**：五号宋体
- [x] **引用格式**：顺序编码制，上标标注，如：[1]
- [x] **著录格式标准**：GB/T 7714-2015《信息与文献 参考文献著录规则》
- [x] **行距**：单倍行距
- [x] **编号格式**：[1], [2], [3]……

---

## 8. 页眉页脚

### 页眉
- [x] **页眉内容**："重庆交通大学本科毕业论文（设计）题目"
  - 从正文第一页开始添加页眉
  - 单面打印时每页相同
- [x] **字体字号**：小五号宋体
- [x] **对齐方式**：居中
- [x] **页眉线**：0.4pt实线

### 页脚
- [x] **页码格式**：阿拉伯数字（1, 2, 3……）
- [x] **字体字号**：小五号宋体
- [x] **对齐方式**：居中
- [x] **起始页**：
  - 封面、声明页不编页码
  - 摘要、目录使用罗马数字（i, ii, iii...）
  - 正文开始使用阿拉伯数字，从1开始

---

## 9. 其他部分

- [x] **原创性声明**：需要，"本科毕业论文（设计）成绩使用声明"
- [x] **版权授权书**：需要，"本科毕业论文（设计）版权使用授权书"
- [x] **致谢**：需要，"致谢"，三号黑体居中，正文小四号宋体
- [x] **附录**：可选，根据需要添加
  - 标题格式："附录A"、"附录B"……
  - 三号黑体，居中
- [x] **外文译文**：需要附在最后（装订于论文后）

---

## 10. 特殊要求

- [x] **文档顺序**：
  1. 封面
  2. 成绩使用声明和版权授权书
  3. 中文摘要
  4. 英文摘要（含英文题目）
  5. 目录
  6. 正文
  7. 参考文献
  8. 致谢
  9. 附录（如有）
  10. 外文译文及原文（装订于最后）

- [x] **引用标注**：
  - 引用处右上角标注，如：……成果[1]。
  - 多篇文献：[1,3,5] 或 [1-3]

- [x] **量和单位**：严格执行GB 3100～3102-93标准

- [x] **数字使用**：
  - 统计数据使用阿拉伯数字
  - 世纪、年代、月、日使用阿拉伯数字
  - 约数使用汉字

- [x] **标点符号**：使用中文标点（全角），英文段落使用英文标点（半角）
