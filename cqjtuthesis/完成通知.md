## 🚀 使用方法

### 第一步：编译提取文档类

```cmd
cd C:\Edu\course\CQJTU-Latex-Beamer\cqjtuthesis
.\makewin.bat thesis
```

或手动：
```cmd
xelatex cqjtuthesis.ins  # 提取.cls文件
xelatex main            # 编译主文档
bibtex main             # 处理参考文献
xelatex main            # 再次编译
xelatex main            # 最终编译
```

### 第二步：修改内容

1. 编辑 `main.tex` 填写封面信息
2. 编辑 `contents/abstract.tex` 填写摘要
3. 编辑各章节文件填写正文
4. 编辑 `ref/refs.bib` 添加参考文献

### 第三步：编译生成PDF

```cmd
.\makewin.bat thesis
```

生成的PDF文件：`main.pdf`

## 💡 重要提示

### ⚠️ 编译要求
- **必须使用 XeLaTeX**（不支持 pdfLaTeX）
- 需要完整的 TeX Live 2020+ 或 MiKTeX 最新版
- Windows 系统自带字体（宋体、黑体、楷体、仿宋）

### 📝 使用建议
1. **首次使用**：先阅读 `QUICKSTART.md`
2. **参考格式**：查看 `FORMAT_REQUIREMENTS.md`
3. **查看示例**：参考 `contents/` 下的文件
4. **清理文件**：使用 `makewin.bat clean` 清理临时文件

### 🔧 常用命令
```cmd
makewin.bat thesis    # 编译论文
makewin.bat clean     # 清理临时文件
makewin.bat cleanall  # 清理所有生成文件
makewin.bat help      # 显示帮助信息
```
